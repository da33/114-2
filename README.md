## 觀看

- 線上版：https://你的帳號.github.io/repo名稱/
- 本地：直接打開 `index.html`

## 操作

| 按鍵 | 功能 |
|---|---|
| ← → / 空白鍵 | 翻頁 |
| 1–9 | 跳頁 |
| Home / End | 第一 / 最後一張 |
| R | 回到第一頁 |

## 部署到 GitHub Pages

1. 建立 repo，把這兩個檔案上傳（`index.html` + `README.md`）
2. Settings → Pages → Source 選 `main` branch → `/ (root)` → Save
3. 等 1 分鐘，網址自動生成
